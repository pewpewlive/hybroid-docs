<Aside type="note">You need to have a computer in order to use Hybroid Live.</Aside>

## Installation

<Tabs syncKey="os">
  <TabItem label="macOS / Linux">
    Run this command in your terminal:

    ```bash
    curl -fsSL https://hybroid.pewpew.live/install.sh | sh
    ```
  </TabItem>
  <TabItem label="Windows">
    Run this command in PowerShell:
    
    ```powershell
    irm https://hybroid.pewpew.live/install.ps1 | iex
    ```
    
    If PowerShell blocks the script, run this once to allow it for your user:
    
    ```powershell
    Set-ExecutionPolicy -ExecutionPolicy Bypass -Scope Process
    ```
  </TabItem>
</Tabs>

Then, run this command in your terminal:

```bash
hybroid -v
```

The console output should look something like this:

```
> hybroid -v
hybroid-live version 0.1.0
```

And voilà! You are ready to create your first project!

## Setting up VS Code

1. Download and install [VS Code](https://code.visualstudio.com/) or a fork like VSCodium, Cursor or Windsurf.

2. Download the Hybroid Live extension from [GitHub](https://github.com/pewpewlive/hybroid-vscode/releases/latest).

3. Press `Ctrl+Shift+P` and type `Install from VSIX`.

The extension will be released to the Visual Studio Code Marketplace in the future.

<Aside type="tip">

Using AI? Improve the workflow by using our provided [llms.txt](https://pewpewlive.github.io/hybroid-docs/llms.txt) file.

Note: since Hybroid is a new and niche language, AI models may not have the best support for it.

</Aside>

## Setting up ppl-utils

Since Hybroid Live is a language for PewPew Live, it requires [ppl-utils](https://github.com/pewpewlive/ppl-utils) to be installed as well.

## Creating the project

Creating a project in Hybroid Live is trivial. Navigate to a directory of your choice and use:

```bash
hybroid init --package hello-world --name "hello, world!" --output out
```

`package` is the name of the package, `name` is the name of the level, and `output` is the output directory.
