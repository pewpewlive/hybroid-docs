This guide will help you get started with Hybroid Live by installing it and creating your first Hybroid Live project.

## Installation

1. Download the latest release from [GitHub](https://github.com/pewpewlive/hybroid-live/releases/latest).

2. Move the downloaded file to a directory of your choice.

3. Go to settings and add the directory to your PATH.

4. Run this command in your terminal:

```bash
hybroid -v
```

The console output should look something like this:

```
> hybroid -v
hybroid-live version 0.1.0
```

And voil√†! You are ready to create your first project!

## Setting up VS Code

1. Download and install [VS Code](https://code.visualstudio.com/) or a fork like VSCodium, Cursor or Windsurf.

2. Download the Hybroid Live extension from [GitHub](https://github.com/pewpewlive/hybroid-vscode/releases/latest).

3. Press `Ctrl+Shift+P` and type `Install from VSIX`.

The extension will be released to the Visual Studio Code Marketplace in the future.

## Creating the project

Creating a project in Hybroid Live is trivial. Use `hybroid init` to initialize the project. You will be asked a few questions while doing so.
